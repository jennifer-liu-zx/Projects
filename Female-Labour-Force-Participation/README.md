## Overview
This project was done for the MOM-NTU Economics Case Competition 2025.

## Description

We were tasked to consider current conditions, conduct an international scans and propose policies to boost female labour force participation rates. We emerged as Champions in the competition.

## In the directory

1. Essay (`Essay_round 1`): essay submitted for Round 1
2. Presentation Deck (`Presentation_final round`): presentation slides used for the Final Round, containing additional plots on top of those seen in the esay
3. Code (`src`): organised code\
   a. Country identitfication (`countryidentification final`): used to perform quantitative simialrity analysis \
   b. Additional plots (`some plots`): used to generate plots seen in the presentation
4. Data (`data`): data compiled from World Bank, OECD, Department of Statistics, Ministry of Manpower MRSD, and more
